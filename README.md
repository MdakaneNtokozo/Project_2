# Project_2

## About project 2
This project will consist of a mobile application that encourage waste management by doing the following:
- It allows households to invite their family members to join the app
- They can set up weekly tasks or challenges that correlate to waste management
- Family members can earn points and be ranked on a leaderboard
- As a result, they can earn rewards at the end of the week. 


## Technologies used
It will make use of the following technologies:
- Flutter for the frontend development
- ASP.NET Core Web API for the RESTful API
- MySQL Workbench with defined SQL table definitions and test data for the backend.
